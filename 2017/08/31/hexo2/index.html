配置(2) hexo deploy 无需输入用户名和密码

每次部署执行hexo deploy都需要输入用户名和密码，很烦

解决方法：

###1.配置好ssh key
在终端（Terminal）输入：

ssh-keygen -t rsa -C “Github的注册邮箱地址”

全部Enter默认即可。
秘钥生成完会得到两个文件id_rsa和id_rsa.pub，路径位于~/.ssh/
用vim打开id_rsa.pub
复制所有内容，然后进入 https://github.com/settings/ssh：
将所复制的内容粘贴至Key的输入框，Title部分的内容随便添加，然后点击Add SSH key。

###2.修改_config.yml
检查deploy那一项，如果repository后面填的是由https://github.com构成的地址，那么请将其改成git@github.com:。

3.重新提交部署

hexo deploy
